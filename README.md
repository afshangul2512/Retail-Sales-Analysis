# Retail Sales Data Analysis

## Project Overview
This project performs data cleaning and exploratory data analysis on a retail sales dataset.

## Objectives
- Clean and preprocess data
- Identify sales trends
- Analyze regional and category performance
- Generate business insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- West region generates the highest sales.
- Sales show seasonal trends.
- Some cities contribute significantly to revenue.
